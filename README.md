## Descriptografar
 - Para descriptogravar, use a [CHAVE](/MINHA_CHAVE) 
 - Execute os seguintes comandos:
```
git-crypt unlock MINHA_CHAVE
```
 - OBS: NUNCA deixe sua chave privada no repositorio! Esta é somente para fins de estudo!
